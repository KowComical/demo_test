# Participant Pilot Entry

This directory is the clean participant-facing entry for the co-speech motion
consistency pilot.

- Public URL target: `/pilot-participant/`
- Default mode: `participant`
- Participant trial count: configured in `data/demo_manifest.json`
- Video assets are reused from `../pilot-demo/assets/videos/` to avoid copying
  the same MP4 files twice.

The review/researcher entry remains in `docs/pilot-demo/`.
